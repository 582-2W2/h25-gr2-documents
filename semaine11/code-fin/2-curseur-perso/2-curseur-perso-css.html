Curseur personnalisé au survol de l'image - Propriété CSS cursor avec url

Propriété CSS d'un curseur personnalisé : cursor:url(urlImage) decalageX decalageY, mot-clé du curseur par défaut;
Les valeurs de decalageX et decalageY permettent de paramétrer le point d'action (hotspot) du curseur par rapport au coin en haut à gauche de l'image.
