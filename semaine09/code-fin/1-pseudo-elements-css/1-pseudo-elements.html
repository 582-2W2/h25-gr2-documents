Notions de base sur les pseudo-éléments

Un pseudo-élément CSS peut être utilisé pour styliser les parties spécifiées d'un élément HTML comme tel.
Comme dans cet exemple, il peut s'agir de styliser la première lettre d'un paragraphe (::first-letter), ou encore sa première ligne (::first-line).

La syntaxe d'un pseudo-élément est la suivante: 

sélecteur::pseudo-élément {
	propriété : valeur;
}

Depuis CSS3, on utilise deux fois le caractère deux-points « :: » pour préfixer les pseudo-éléments (afin de pouvoir les différencier des pseudo-classes). Toutefois, la plupart des navigateurs prennent en charge les pseudo-éléments pour lesquels le préfixe n'est formé que d'un seul « : ».

Les pseudo-élements peuvent aussi être utilisés pour ajouter du contenu cosmétique avant (::before) ou après (::after) le contenu actuel d'un élément HTML. Pour cela, on doit utiliser et définir la propriété CSS:content. L'élément ajouté peut être du texte, un symbole graphique, un icône UTF8, une image, etc. La syntaxe de base sera alors la suivante: 

sélecteur::pseudo-élément {
	content : valeur;
}

Mettre un élément avant et après avec ::before et ::after

Les pseudo-éléments générés par ::before et ::after sont contenus dans la boîte de mise en forme de l'élément: ::before comme étant le premier enfant de l'élément et ::after comme le dernier enfant de cet élément. Par défaut, l'élément créé est de type en-ligne (inline). Aussi, les pseudo-élements ne sont pas compris avec le sélecteur CSS universel (*). Ainsi, il faudra s'assurer de leur appliquer la même normalisation que les autres éléments HTML: 

::before, ::after {
	margin: 0;
	padding: 0;
	box-sizing: border-box;
}

Ajout et stylisation de contenu vide

Il est aussi possible d'ajouter un contenu, comme une boîte vide en utilisant (::before ou ::after) et de la formater. Pour cela, l'élément HTML doit avoir une position CSS définie autre que celle du flux régulier du document (static), habituellement: position:relative. Et, celle du pseudo-élément doit être en absolu: (position: absolute) afin d'être en mesure d'ajuster sa position à celle de l'élément parent. Le contenu vide est exprimé comme suit: 

sélecteur::pseudo-élément {
	content : "";
}
